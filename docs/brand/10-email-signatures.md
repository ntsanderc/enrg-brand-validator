### Gmail

1. Open Gmail â†’ Click **Settings** (gear icon) â†’ **See all settings**

3. Click **+ Create new** â†’ Name it "ENRG Signature"

### Outlook (Desktop)

1. Open Outlook â†’ **File** â†’ **Options** â†’ **Mail** â†’ **Signatures**

3. Paste HTML code (use **Insert as HTML** if available)

### Apple Mail

1. Open Mail â†’ **Mail** â†’ **Settings** â†’ **Signatures**

4. Name it "ENRG Signature"
5. Paste HTML code (may need to use **Edit â†’ Paste and Match Style**)
6. Drag-and-drop headshot image into signature area

### Outlook (Web)

1. Open Outlook.com â†’ **Settings** (gear icon) â†’ **View all Outlook settings**
2. Go to **Mail** â†’ **Compose and reply**

5. Check "Include signature on new messages" and "on replies/forwards"
6. Click **Save**

## Email Content Best Practices

### Subject Lines

**Good:**
- "New Listing: 3BR Home in [Neighborhood] - $XXX,XXX"
- "Market Update: [City] Real Estate Trends - January 2026"
- "Open House This Weekend - [Address]"

**Avoid:**
- ALL CAPS SUBJECT LINES
- Excessive punctuation (!!!)
- Misleading clickbait

### Greeting

**Professional:**
- "Hi [First Name],"
- "Hello [First Name],"
- "Good morning/afternoon, [First Name],"

**Avoid:**
- "Hey" (too casual)
- "Dear Sir/Madam" (too formal)
- No greeting (cold)

### Closing

**Recommended:**
- "Best regards,"
- "Best,"
- "Thank you,"
- "Looking forward to hearing from you,"

**Avoid:**
- "Cheers" (too casual)
- "Sincerely yours" (too formal)
- No closing (abrupt)

## QC Checklist

Before sending email, verify:

- [ ] Signature includes correct headshot
- [ ] All links are clickable and lead to correct URLs
- [ ] Phone number is formatted consistently
- [ ] Email address is correct
- [ ] Logo displays properly (not broken image)
- [ ] Equal Housing Opportunity disclosure present (agents only)
- [ ] Mobile preview looks acceptable (test on phone)
- [ ] Spelling and grammar are correct
- [ ] Recipient name is spelled correctly
- [ ] Subject line is clear and accurate
